1.Converter de binário para decimal

1.1

0000 -> 0

1.2

1011 -> 1\*2^0 + 1\*2 + 1\*2^3 = 1 + 2 +8 = 11

1.3

11111111 -> 255

1.4

0101 -> 1 + 1\*2^2 = 5

1.5

11111 -> 2^0 + 2^1+ 2^2 + 2^3 +2^4 = 1 + 2 + 4+ 8 + 16 = 31

1.6

10101010 -> 1\*2^1+1\*2^3+1\*2^5+1\*2^7 = 2 +8 + 32 + 128 = 170

1.7

100001 ->1 + 1\*2^5 = 33

1.8

1001 -> 1 + 2^3 = 9
